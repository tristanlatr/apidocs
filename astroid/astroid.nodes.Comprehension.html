  • Term presence. The below example searches for documents that must contain “foo”, might contain “bar” and must not contain “baz”:
    '+foo bar -baz'
  • Wildcards. The below example searches for documents with words beginning with “foo”:
    'foo*'
  • Search in specific fields. The following search matches all documents that contains "foo" in their full dotted name and with “bar” in any field:
    '+fullName:*foo* +bar'

astroid.nodes.Comprehension

class Comprehension(NodeNG): (source)

View In Hierarchy

Class representing an :class:`ast.comprehension` node.

A :class:`Comprehension` indicates the loop inside any type of comprehension including generator expressions.

>>> import astroid
>>> node = astroid.extract_node('[x for x in some_values]')
>>> list(node.get_children())
[<Name.x l.1 at 0x7f23b2e352b0>, <Comprehension l.1 at 0x7f23b2e35320>]
>>> list(node.get_children())[1].as_string()
'for x in some_values'
