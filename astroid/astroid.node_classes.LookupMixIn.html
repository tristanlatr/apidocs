def lookup(self, name): (source)

Lookup where the given variable is assigned.

The lookup starts from self's scope. If self is not a frame itself and the name is found in the inner frame locals, statements will be filtered to remove ignorable statements according to self's location.

ParametersnameThe name of the variable to find assignments for. (type: str)
ReturnsThe scope node and the list of assignments associated to the given name according to the scope where it has been found (locals, globals or builtin). (type: tuple(str, list(NodeNG)))
